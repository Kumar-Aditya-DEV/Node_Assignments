# Student CGPA API

## Objective
Build a REST API using Express.js that manages student CGPA records stored in an in-memory JSON array.

## Implemented Routes

GET /students  
GET /students/topper  
GET /students/average  
GET /students/count  
GET /students/:id  
GET /students/branch/:branchName  

## Sample API URLs

/students  
/students/topper  
/students/average  
/students/count  
/students/1  
/students/branch/CSE  

## Steps to Run Locally

npm install  
node index.js
